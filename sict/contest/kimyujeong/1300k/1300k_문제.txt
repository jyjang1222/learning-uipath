
. 파일 경로, 결과 예시
	-기본 경로 : "C:\temp"경로에 "1300k" 폴더를 생성하고 폴더 안에 "template.xlsx" 파일 저장
	-결과 예시 : "예시.xlsx" 참조
	- c:\temp\1300k폴더안의 "template.xlsx" 엑셀파일을 복사하여 c:\temp\1300k폴더에 "yyyy년 MM월 dd일 플래너 검색 결과.xlsx" 형식으로 바꾼 엑셀파일에 데이터를 저장해주세요.(ex.'2023년 06월 13일 검색결과.xlsx')

	
. 문제
	- 1300k 사이트에 접속해( "https://www.1300k.com/" ) 플래너에 대해 검색하고 "상품리뷰순"으로 정렬한 후 목록 데이터, 상세페이지 데이터를 추출, 가공 하여 메일로(gmail) 발송


. 파일 관련 처리
	- 결과 파일은 "template.xlsx"파일을 복사해  "플래너 결과.xlsx"로 저장해 주세요 - Copy File Activity 사용

		 
. 제약사항
	- 데이터 추출 관련
		- 검색 데이터는 210개를 추출한 뒤 가공.
		- 검색 데이터 중 sold out이 아니고 호감도가 1개 이상인 경우만 결과 데이터에 저장해 주세요.

 
	- 검색 결과 엑셀파일 관련
		- 할인가가 없는 상품은 원가를 할인가에 표시한다
		- 할인율이 없을 경우 "0%"로 표시
		- 할인가 기준으로 정렬 후 별점 기준으로 정렬
		- 최저가가 동일한 경우 먼저 추출한 게시물을 최저가 게시물로 처리
		- 최대 할인율이 동일한 경우 먼저 추출한 게시물을 최대 할인율 게시물로 처리
		
	- 메일 발송 관련	
		- SMTP Activity 사용 금지
		- 별점이 5인 게시글 중 최저가 게시물을 메일 본문에 입력
		
	- Workflow 관련

		
. 메일 양식
		- 수신자 : <본인 gmail계정>
		- 제목 : 1300k "yyyy년 MM월 dd일" 플래너 검색내용 
		- 본문 :
			별점이 5인 게시글 중
			
			최저가 게시물 상품명: 
			최저가 게시물 url: 
			최저가 게시물 브랜드: 
			최저가 게시물 원가: 
			최저가 게시물 할인가: 
			최저가 게시물 할인율: 
			최저가 게시물 배송비: 
			최저가 게시물 리뷰수: 
			최저가 게시물 호감도수: 
		
		-첨부파일 : "<yyyy년 MM월 dd일> 플래너 검색결과.xlsx" 파일

		
. 결과물
	- "yyyy년 MM월 dd일 플래너 검색 결과.xlsx" 파일 
	- Workflow 파일
	- 수신한 메일
