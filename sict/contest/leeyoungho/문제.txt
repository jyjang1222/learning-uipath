문제 개요.
	- 텐바이텐 사이트에서 특정 데이터를 수집하여, 메일발송.
	
- 기호에 대한 설명.
	*<>는 변수를 뜻합니다.
	*[] 는 데이터 수집목록의 컬럼명을 뜻합니다.
	
- 작업파일 경로생성 및 규칙
	- 프로젝트 data폴더의 input폴더,result폴더 를 복사하여 c:\temp\텐바이텐 경로로 복사합니다.
	- c:\temp\텐바이텐\input폴더안의 "탬플릿.xlsx" 엑셀파일을 복사하여 c:\temp\텐바이텐\result폴더에 "yyyy년 MM월 dd일 검색결과.xlsx" 형식으로 바꾼 엑셀파일에 데이터를 저장해주세요.(ex.'2023년 03월 10일 검색결과.xlsx')
	

	
- 참고사항
	* 작성예시는 c:\temp\텐바이텐\result폴더의 "작성예시.xlsx" 파일을 참고해주세요.

- 데이터 수집
	
	- 데이터 위치
		* 텐바이텐 ( https://www.10x10.co.kr/ ) =>디자인 문구 => 데코레이션 => 스티커 => 인기상품순 정렬
		* 목록 및 제품 상세 페이지
		
	- 데이터 수집 목록
		* 수집할 데이터 목록은 input폴더안의 "탬플릿.xlsx" 엑셀파일을 참고해주세요.
		
	- 데이터 수집 범위
		* 1page ~ 11page
		
	- 데이터 수집 규칙     
		* [할인가],[할인율]은 없을 경우 따로 작업을하지않습니다.
		* 리뷰수나, 좋아요갯수가 목록과 상세페이지가 다를경우 목록에 표시된값을 사용해주세요
- 스크린샷 규칙
		*  제조사,상품명,판매가,제품이미지를 포함하여야합니다.

- 메일 발신
	* Gmail에 접속하여 발신해주세요. (SMTP액티비티를 사용하지않습니다.)
	* 메일의 수신인은 자기자신에게
	* 메일제목: yyyy년 MM월 dd일텐바이텐 스티커 1~<N>위 검색 결과입니다.
	* 메일본문: 좋아요가 제일 많은제품 : "좋아요가 제일 많은제품의 제품명"
	      	 좋아요 갯수 : "좋아요가 제일 많은제품의 좋아요 갯수"
	* 첨부파일 : "가장 좋아요가 많은 제품의 화면 스크린샷"파일, c:\temp\텐바이텐\result폴더의 "yyyy년 MM월 dd일 검색결과.xlsx"파일
