● 사전 준비(사람의 영역)
  - "funcTemplate.xlsx" 파일을 "C:\template\" 경로에 넣는다

● 파일 경로 및 이름
  - 템플릿 경로: "C:\template\"
  - 결과출력 경로: "C:\result\"
  - 결과파일 이름: "엑셀기초함수.xlsx"
  - 엑셀 시트 이름: "Sheet1"

● 사이트 주소
  "https://www.oppadu.com/엑셀-함수-목록/"

● 사전지식
  - 엑셀 함수 안에 콤마(",")를 기준으로 나뉜 각각의 요소들을 "인수"라 한다
  - = 함수명(): 인수가 최대 0개 들어간다
  - = 함수명(인수1, 인수2): 인수가 최대 2개 들어간다
  - = 함수명(인수1, [인수2]): 인수가 최대 2개 들어간다
  - = 함수명(인수1, [인수2], ...): 인수가 최대 255개 들어간다

  - [name]: name은 컬럼명(헤더)

  - dataTable.AsEnumerable.Where... 이렇게 시작하는 것은 LINQ 메소드 구문
  - (From row In dataTable.AsEnumerable... 이렇게 시작하는 것은 LINQ 쿼리 구문
  - 두 구문 다 표현 방식의 차이만 있고 결과값은 똑같다(성능의 경우는 모르겠음)

● 문제
  - "funcExample.xlsx"를 참고하여 각 컬럼들을 채워 넣는다
  - [범주]를 기준으로 오름차순, [댓글수]를 기준으로 내림차순 정렬한다

  - [범주]: 엑셀 함수가 포함된 그룹 // 출력예시: 날짜 및 시간
  - [함수명]: 엑셀 함수의 이름 // 출력예시: DATEDIF
  - [구문]: 엑셀 함수 사용법 // 출력예시: DATEDIF ( 시작일, 종료일, 단위 )
  - [최대인수개수]: 엑셀 함수가 최대로 가질 수 있는 인수의 개수 // 출력예시: 3
  - [댓글수]: 엑셀 함수 상세페이지에 적힌 댓글의 수 // 출력예시: 12
  - [수량]: 해당 범주의 함수의 수량, 예를 들어 [범주]가 "날짜 및 시간"인 함수의 개수는 20개이다 // 출력예시: 20
  - [최다댓글함수]: 해당 범주에서 가장 많은 댓글을 받은 함수, 예를 들어 [범주]가 "날짜 및 시간"이고 [댓글수]에서 가장 많은 댓글을 받은 함수는 "DATEIF"이다 // 출력예시: DATEDIF

● 함정사항
  - TEXTJOIN 함수 상세 페이지의 경우만 셀렉트가 다르게 잡히는 때문에, 이 하나의 경우만 따로 분기한다

● 권장사항
  - 반복문 3개 이하 사용
  - Get Text 6개 이하 사용

● 제약사항
  - 변수 형식(자료형)이 "DataTable"인 변수는 3개 이하를 사용한다
  - "Add Data Row" 액티비티 1개 이하를 사용한다
  - Write Range Workbook의 결과로 엑셀 셀 좌측 상단에 녹색 삼각형이 뜨게 하지 않는다
  - 엑셀에 지정 되어 있는 기존의 서식을 깨지 않는다
  - [함수명]에서 중복값이 있으면 안 된다
  - 모든 컬럼은 빈 값이 존재하지 않는다
  - 다중정렬 시 "Sort Data Table" 액티비티는 사용하지 않는다
  - [최대인수개수], [수량], [최다댓글함수] 중 적어도 하나는 LINQ 구문을 사용해야 한다

● 수정사항
  - 변수 형식(자료형)이 "DataTable"인 변수는 2개 이하를 사용한다 ▷ 변수 형식(자료형)이 "DataTable"인 변수는 3개 이하를 사용한다
