* [문제]
    오늘의집 https://ohou.se/ 에 접속해 입력받은 <검색어> 검색 결과 목록 데이터 및 상세페이지 데이터 추출 및 가공 후 메일(Gmail) 발송

* 문제에 사용된 기호 설명
	- <>: 표시할 변수의 값

* 파일경로 및 파일명 세팅 참조
    ▼ 기본 파일 경로는 아래와 같이 세팅한 후 진행해 주세요 ▼
    - 기본경로 : C:\오늘의집
    - template 파일경로 : C:\오늘의집\오늘의집_template.xlsx

    ▼ output 파일 경로는 uipath에서 아래와 같이 세팅해 주세요 ▼
    - output 폴더경로 : C:\오늘의집\<검색어>
        output 폴더는 Creat folder Activity를 사용하여 세팅해 주세요.
    - output excel 파일명 : "yyyy-MM-dd <검색어> 검색결과.xlsx"
        output excel 파일은 Copy file Activity를 사용하여 오늘의집_template.xlsx을 복사한 뒤 사용해 주세요.

* 참조 파일
    - "예시.xlsx" 파일 참조
    - "Bread crumb list.png" 파일 참조
    - "메일예시.png" 파일 참조

* 수집해야 할 데이터
    1. 카테고리(bread crumbs list에서 가장 마지막(가장 우측의) depth 아이템만 수집)
    2. 브랜드(브랜드 이름)
    3. 상품코드
    4. 상품명
    5. 상품 url
    6. 정가
    7. 할인가(할인하지 않는 상품일 경우 정가 기입)
    8. 할인율("%" 기입필수)
    9. 별점
    10. 오늘출발 여부(오늘출발 상품이면 "O" 기입, 아니면 "X" 기입)
    11. 도착 예정일(도착 예정일이 없을 경우 "알 수 없음"으로 기입)

* 제약사항
    - 데이터 수집 관련
        → 무료배송인 상품들의 데이터만 수집해 주세요.
        → <maxNum> 변수의 디폴트값은 30 입니다.
        → 검색 데이터는 최대 <maxNum>개 추출해 주세요.
        → 검색된 상품의 전체 개수가 <maxNum>개 미만이면 프로세스를 종료해 주세요.
        → 상품의 url은 꼭 "Data Scraping" 기능을 사용해서 수집해 주세요.
        → "Data Scraping" 기능으로 추출해야 하는 범위는 가장 왼쪽 상단에 있는 제품부터(첫 번째 상품부터) <maxNum>번째 제품까지입니다.
        → "Data Scraping" 기능을 사용하지 않고 완성했을 시 정답이 인정되지 않습니다.

    - 데이터 가공 관련
        → 할인율 평균은 소수점 둘째 자리에서 반올림 처리해 주세요.
        → 수집한 데이터는 1. 할인가 오름차순, 2. 오늘출발 여부 오름차순으로 정렬한 뒤 엑셀에 기입해 주세요.
        → 최저가 상품 데이터를 추출할 때, 최저가가 동일할 경우 먼저 추출한 상품을 최저가 상품으로 처리합니다.
        → 별점이 가장 높은 상품 데이터를 추출할 때, 별점이 동일할 경우 먼저 추출한 상품을 별점이 가장 높은 상품으로 처리합니다.

    - 메일 발송은 Web Gmail을 사용해 주세요.

* 기본 가이드
    1. <maxNum>의 값은 Open browser Activity를 사용하기 전에 Assign Activity를 사용하여 선언해 주세요.
    2. Input dialog Activity 사용하여 <검색어>를 사용자로부터 입력받습니다.
    3. 오늘의집 페이지에 접속합니다.
    4. "판매순"으로 정렬된 검색 결과에서 "무료배송"인 상품들의 url들을 "Data Scraping" 기능을 사용하여 수집합니다.
    5. 상품코드는 상품 url을 활용하여 수집합니다.
    6. 카테고리 데이터는 상품 상세페이지 내 좌측 상단의 *Bread crumb list(Bread crumb list.png 파일참조)와 Find Children Activity를 활용해서 추출해주세요.
    7. 수집한 데이터를 template파일을 활용하여 xlsx 파일로 추출해 주세요.

* 메일 작성 예시
    - 수신자 : 본인
    - 제목 : "[오늘의집] yyyy-MM-dd <검색어> 검색결과"
    - 본문 : 
            1. 별점이 가장 높은 상품의 브랜드명 :
            2. 별점이 가장 높은 상품의 상품명 :
            3. 별점이 가장 높은 상품의 할인가 :

    - 첨부파일 : "yyyy-MM-dd <검색어> 검색결과.xlsx"
