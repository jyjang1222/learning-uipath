. 파일 경로, 결과 예시
	-기본 경로 : 바탕화면에 result폴더 생성 후 "template.xlsx" 파일 저장
	-결과 예시 : "예시.xlsx" 참조
	
. 문제
	- 당근마켓 사이트에 접속해( "https://www.daangn.com/" ) <검색어>에 대한 목록 데이터, 상세페이지 데이터를 추출 한 뒤 가공 하여 메일로(gmail) 발송

. 파일 관련 처리
	- 결과 파일 저장을 위해 기본 경로 하위에 "당근마켓 <검색어>"폴더를 생성해 주세요. - Create Folder Activity 사용
	- 결과 파일은 "template.xlsx"파일을 복사해 "당근마켓 <검색어>" 폴더에  "<검색어> 결과.xlsx"로 저장해 주세요 - Copy File Activity 사용
	- 결과 데이터 중 가격이 0원을 초과하면서 가장 저렴한 게시물의 url로 이동하여 mhtml파일을 "당근마켓 <검색어>" 폴더에 "<검색어> 최저가.mhtml"로 저장해 주세요.
		 
. 제약사항
	- 데이터 추출 관련
		- 중고거래 항목이 존재할 때 까지 검색어를 사용자로부터 입력 받아 검색창에 입력하세요. - Input dialog Activity 사용
		- 검색 데이터는 최대 30개 추출해 주세요.
		- 검색 데이터 중 매너온도가 37도 이상일 경우에만 결과 데이터에 저장해 주세요.
		- 검색한 게시물 개수가 10개 미만이거나 매너온도가 37도 이상인 게시물이 5개 이상 존재하지 않을 시 프로세스 종료
		- 최저가가 동일한 경우 먼저 추출한 게시물을 최저가 게시물로 처리
 
	- 검색 결과 엑셀파일 관련
		- 가격, 좋아요 항목이 존재하지 않을 시 "없음" 으로 표시
		- 끌올여부 내림차순, 가격 오름차순으로 정렬  
			( 가격이 "없음"일 경우 0원으로 처리,  끌올여부는 O, X로 표시 )
		- 가격 표시 통일 ( 백만원까지 ) 100만원 -> 1,000,000원, 100만2000원 -> 1,002,000원
		- 최저가, 최대 좋아요, 가격총합, 가격평균은 가격, 좋아요 "없음"을 제외한 값에서 추출
		- 가격 평균은 소수점 두 번째 자리에서 반올림
		
	- 메일 발송 관련	
		- SMTP Activity 사용 금지
		
	- Workflow 관련
		- Workflow내  반복관련 Activity는 3개 까지만 사용해 주세요.
		- 천자리 쉼표 처리는 UiPath내에서 처리해주세요.
